Modes

Editor uses modes to...
Available modes:

Mode name Activation key Short description Notes
Normal ESCAPE Just normal mode -
Insert I Insert new curves. Click on canvas to add control points. Pressing shift and clicking will end curve -
Move M Move points around the canvas -
Delete D Delete points -
Append A Adds point after selected point in a curve. If no point is selected new point will be added at the end of the selected curve If no curve or point is selected error will be displayed
Prepend P Adds point before selected point in a curve. If no point is selected new point will be added at the beggining of the selected curve If no curve or point is selected error will be displayed
